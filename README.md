# raahul42.github.io

Personal portfolio site — a story-driven single-page site rather than a resume dump.

## What's on it

- **Hero** — typewriter cycling through roles (some serious, some not)
- **My Story** — a short narrative about the journey from Chemical Engineering to Data Science
- **Journey timeline** — NIT Trichy → Capgemini → IIM Calcutta → Data Scientist (2024–present)
- **What I'm Building** — fraud prevention work professionally; tree-tracker and YouTube content personally
- **YouTube embed** — latest video
- **Easter egg** — 🎭 Vadi Mode

## Stack

Pure HTML, CSS, and vanilla JS. No frameworks, no build step, no dependencies beyond a couple of CDN fonts and icons. Hosted on GitHub Pages.

Analytics via [GoatCounter](https://www.goatcounter.com/) — privacy-friendly, no cookies.

## How it was built

Started with a rough brief, then iterated with [Claude](https://claude.ai/claude-code) — the redesign, the story copy, the CSS theming system (dark/light mode with a warm cream palette for light), and all the small details like the nav tooltips and mobile card layout. Went through several rounds of "make it less resume-like" until it felt right.

The `& claude` in the footer credit is intentional.
